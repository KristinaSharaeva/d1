# Django Project

Это проект на Django, содержащий два представления:
- Главная страница
- Страница "О себе"

## Установка

Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```bash
## Зависимости
pip install django

## Запуск
python manage.py runserver



 
Логи посещений страниц сохраняются в файл debug.log.
